Vista de generos

Esta pantalla cuenta con:

  • Lista de géneros
  • Campo para insertar nuevo género
  • Botón para crear un nuevo género
  • Botón para borrar el género seleccionado

Debajo del encabezaod de la pantalla hay un desplegable que contiene los géneros
cuando hay uno seleccionado, se puede hacer uso del botón de "Borrar" que lo eliminará y recargará la lista
debajo del desplegable se encuentra una caja de texto donde se podrá poner el nombre de un género,
que no exista y que tenga entre 1 y 255 caracteres, para poder insertarlo haciendo uso del botón "Nuevo"
