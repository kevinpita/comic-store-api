Vista de informes

Esta pantalla es muy simple. Permite visualizar los 4 tipos de informes que se pueden obtener a partir de los datos
El primero de ellos, Lista de películas, abriría un informe de las peliculas que hay guardadas.

El segundo, Lista de actores, haría lo mismo que el primero, pero en este caso el informe sería de actores.
El tercero, Películas por género, permitiría obtener un informe con la lista de películas que hay guardadas del género seleccionado en el desplegable.
El cuardo, Filmografía de un actor, permitiría obtener un informe con la lista de películas donde ha participado un actor.
