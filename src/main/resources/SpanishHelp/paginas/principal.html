Gestión de Películas

Esta es la pantalla principal de la aplicación.

Desde aquí se puede:

  • Conectar a la base de datos
  • Modificar la conexión
  • Cambiar el idioma
  • Abrir la ayuda
  • Abrir la vista de películas
  • Abrir la vista de actores
  • Abrir la vista de generos
  • Abrir la vista de informes

Esta pantalla cuenta con un menú superior y con 5 botones centrales.

En el menú superior hay 4 apartados diferentes

  • Conexión: permite seleccionar si se usará la conexión por defecto o la del archivo de configuración
  • Configuración: permite cambiar el idioma (español e inglés) y editar la conexión del
  • Herramientas: permite crear datos base y crear la estructura en la base de datos
  • Ayuda: permite abrir la ayuda

La configuración por defecto es:

  • Nombre base de datos: gestion_peliculas
  • Dirección: localhost
  • Puerto: 3306
  • Usuario: root
  • Contraseña: root

Hay ciertas acciones que solo se pueden hacer cuando se está conectado:

  • Abrir vista de películas
  • Abrir vista de actores
  • Abrir vista de géneros
  • Abrir vista de informes
  • Crear datos

También hay ciertas acciones que solo se pueden hacer cuando se está desconectado:

  • Cambiar el método de conexión
  • Editar la configuración de la conexión
  • Crear estructura de la base de datos
